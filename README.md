# Promise

Promise library for Roblox, packaged for pesde.

## Install

```bash
pesde add gh#idkaboutbytes/promise#v2.0.0
pesde install
```

## Require

```lua
local Promise = require(game.ReplicatedStorage.Packages.Promise)
```

## Publish

```bash
pnpm run packages:release -- promise
pnpm run packages:publish -- promise --yes
```
